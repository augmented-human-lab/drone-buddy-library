Voice Recognition Module Installation
=============================================

Currently DroneBuddy supports several algorithms

    #.  VOSK - Offline package
    #.  Google - Wrapper for OpenAi
    #.  Multi Algorithm support - Wrapper for third party library `SpeechRecognition <https://pypi.org/project/SpeechRecognition/>`_

To use each of these you can customize the installation
